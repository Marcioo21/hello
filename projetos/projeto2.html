Descrição do Projeto

Este projeto é uma aplicação web que utiliza HTML, CSS e JavaScript. O objetivo é criar uma interface responsiva que permite ao usuário interagir de forma intuitiva. Meu segundo projeto para um grupo de pessoas. Demorei poucas semanas para fazer e entregar ao propietário. Um site responsivo para qualquer despositivo. um site de uma organizaão de um jogo RP. Foi usado apenas HTML, CSS E JavaScript. Esse site não tem banco de dados por opção do propietário.

Tecnologias Utilizadas

  • HTML
  • CSS
  • JavaScript
